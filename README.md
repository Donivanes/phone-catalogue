# phone-catalog-app

## To run the application

Clone the application:

```
git clone https://github.com/Donivanes/phone-catalogue.git
```

Run the server:

```
cd server/
npm install
npm run dev
```

Run the client:

```
cd client/
npm install
npm start
```

## To run with docker

Go to the root of the app and run:

```
docker-compose up
```

or

```
sudo docker-compose up
```
